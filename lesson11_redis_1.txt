-- Lesson 11. REDIS & MongoDB.

-- task 1. В базе данных Redis подберите коллекцию для подсчета посещений с 
-- определенных IP-адресов.

-- Этапы решения:
-- 1. Команда SADD позволяет вставлять в коллекцию сразу несколько значений:

SADD ip '192.168.0.1' '192.168.0.2' '192.168.0.3' '192.168.0.4' '192.168.0.5' '193.168.0.1' '193.168.0.2' '193.168.0.3' '194.168.0.1' '194.168.0.2'

-- 2. С помощью SCARD подсчитываем число ip адресов:

SCARD ip
